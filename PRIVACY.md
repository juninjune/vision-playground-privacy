# Vision Playground - 개인정보 처리방침

## 정보 수집

Vision Playground는 어떠한 개인정보도 수집하지 않습니다.

## 데이터 처리

- 모든 OCR 처리는 사용자의 Mac에서 로컬로 수행됩니다
- 이미지 및 PDF 파일은 기기 밖으로 전송되지 않습니다
- 인터넷 연결이 필요하지 않습니다
- Apple Vision Framework를 사용한 완전한 로컬 처리

## 데이터 저장

- 사용자가 명시적으로 선택한 파일만 읽습니다
- JSON 결과는 사용자가 지정한 위치에만 저장됩니다
- 앱은 어떠한 데이터도 외부 서버로 전송하지 않습니다
- 앱 종료 시 처리된 데이터는 자동으로 삭제됩니다

## 권한

앱은 다음 권한을 요청합니다:
- **파일 읽기/쓰기**: 사용자가 선택한 이미지/PDF 파일 읽기 및 JSON 결과 저장용

## 개발자 도구

이 앱은 개발자가 Apple Vision Framework의 RecognizeTextRequest API 응답 구조를 확인하기 위한 도구입니다. 일반 사용자용이 아닙니다.
