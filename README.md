# Vision Playground - 개인정보 처리방침

## 정보 수집
Vision Playground는 어떠한 개인정보도 수집하지 않습니다.

## 데이터 처리
- 모든 OCR 처리는 기기 내에서 로컬로 수행됩니다
- 이미지 및 PDF 파일은 기기 밖으로 전송되지 않습니다
- 인터넷 연결이 필요하지 않습니다

## 데이터 저장
- 사용자가 선택한 파일만 읽습니다
- JSON 결과는 사용자가 지정한 위치에만 저장됩니다
- 앱은 어떠한 데이터도 서버로 전송하지 않습니다
